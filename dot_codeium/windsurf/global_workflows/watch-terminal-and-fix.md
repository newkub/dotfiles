1. ดู terminal ที่เปิดอยู่ ห้าม run ใหม่
2. แก้ไข error, warning ให้หมดจนไม่มีเหลือ
3. ทำซ้ำ ห้ามออก watch ต่อไปยาวๆเลย ถ้าจะออกผมกดออกเอง แต่คุณห้ามหยุด watch ต่อไปยาวๆเลย


หมายเหตุ
- แค่ watch แล้วแก้ไขอย่างเดียว ห้าม run lint, run dev, run build เอง
- ห้ามออก ให้ watch ต่อไปเรื่อยๆ ไม่ต้องออก ผมจะบอกให้ออกเอง แต่คุณห้ามออกเอง

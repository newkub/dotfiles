1. run lint ใน package.json
2. เอา error, warning มัน planing และ prioritize ว่าต้องแก้อะไรบ้าง และแก้อะไรก่อน
3. แก้ตามที่ plan ให้ครบ
4. run lint ซ้ำ จนกว่าจะไม่มี warning, error อะไรอีกแล้ว


หมายเหตุ : ให้ทำงานได้เลยโดยไม่ต้องถาม

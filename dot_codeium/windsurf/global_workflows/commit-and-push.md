## 1. การ Commit (ใช้เสมอ)

1.1. ทำ /commit เพื่อ commit ไฟล์ที่เปลี่ยนแปลง

## 2. การ Push (ใช้เมื่อ commit เสร็จสิ้น)

2.1. Push commit ไปยัง remote repository
   - git push

2.2. ถ้า push ไม่สำเร็จ ให้ทำข้อ 2.1 ใหม่จนกว่าจะผ่าน
